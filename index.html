Centralized Terminal Ticketing System

Terminal Centralized Ticketing System, this system is designed to manage ticket selling and tracking in a centralized way. The system works by allowing staff to create and manage tickets, record transactions, and store all ticket data in a centralized database. This helps ensure consistency and makes reporting easier.

One challenge I faced was ensuring data accuracy and preventing duplicate or incorrect entries. To address this, I implemented input validation and structured the database properly to maintain data integrity.

From developing the Terminal Centralized Ticketing System, I learned the importance of maintaining data accuracy in transaction-based systems. I improved my skills in designing structured databases to avoid duplication and ensure consistency, and I gained experience in implementing input validation to prevent errors. Overall, the project helped me understand how critical reliable data handling is in building effective real-world systems.

Smart Store

SmartStore is a modern web-based shopping platform for browsing and ordering food products online. It features category filtering, a responsive design, and an intuitive interface that allows users to easily find items and manage their cart. Additionally, the system includes a built-in chat functionality that enables customers to communicate directly with the admin for inquiries, concerns, or support, enhancing overall user experience and customer service.

One challenge I faced was building a customer support chat that gives quick answers. To solve this, I created a structured reply system where the admin can pre-set responses to common questions, and added quick-reply buttons so customers get instant answers without typing.

From building SmartStore, I learned how to develop a real-time chat system between customers and the admin. I gained experience in structuring conversations in a database, displaying messages dynamically without page refresh, and building role-based views for both the customer and admin side.

Insightify

Insightify is web-based reporting system that transforms raw transaction data into meaningful business insights. It features dynamic data visualization using interactive charts, real-time filtering, and detailed reports for sales trends, product performance, and user activity. The system helps administrators make data-driven decisions by providing clear and actionable analytics.

While developing my website, I encountered 404 and 403 errors and issues with frontend, backend, and database integration. I used browser developer tools (F12) to debug and identify problems such as incorrect file paths and server configuration issues. I resolved them by fixing the directory structure, broken links, and database connection, which improved my understanding of web development.

What I learned from making this website is not only how to utilize the resources I have, such as developer tools, but also how to create a website that is user-friendly and useful, especially in solving real-world problems.
